# Gerber Output
*These files can be sent directly to the PCB house*
### Gerber file extensions are:
|Extension| Description|
|---------|------------|
|GBL| Bottom Silkscreen|
|GBO| Bottom Copper|
|GBS| Bottom Soldermask|
|GTL| Top Copper|
|GTO| Top Silkscreen|
|GTS| Top Soldermask|
|TXT| Drill File|
|dri| Drill Station Info File|
|gpi| Photplotter Info File|
|GML|Mill Layer|
|GTP|Top Paste|
